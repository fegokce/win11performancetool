# 🚀 Win11 Debloat Tool

> ⚠️ **Önemli / Important**
> Bu aracı kullanmadan önce `.py` dosyasını build etmeniz gerekmektedir.
> Before using this tool, you must build the `.py` file.

```bash
pyinstaller --onefile --console --uac-admin --name "Win11Debloat" --icon NONE win11_performance.py
```

---

## 🇹🇷 Türkçe

### 📌 Proje Hakkında

**Win11 Debloat Tool**, Windows 11 sistemlerde:

* Gereksiz uygulamaları kaldırır
* Telemetri ve veri toplama işlemlerini kapatır
* Performans optimizasyonu yapar
* Windows Update ayarlarını kontrol altına alır

CLI (komut satırı) tabanlıdır ve **Administrator yetkisi gerektirir**.

---

### ⚙️ Özellikler

#### 🔒 Telemetri & Gizlilik

* Telemetri servislerini kapatır
* Cortana devre dışı bırakılır
* Reklam ID kapatılır
* Activity History devre dışı
* Konum servisi kapatılır

#### 🧹 Bloatware Temizleme

* Xbox uygulamaları
* Teams, Skype
* Candy Crush, TikTok vb.
* Gereksiz Microsoft uygulamaları

#### ⚡ Performans Optimizasyonu

* Yüksek performans güç planı
* Gereksiz servisleri kapatma
* Animasyonları devre dışı bırakma
* Disk ve sistem optimizasyonları

#### 🔄 Windows Update Kontrolü

* Otomatik restart kapatma
* Driver update kapatma
* Store auto update kapatma

---

### ▶️ Kullanım

1. Programı **Yönetici olarak çalıştır**
2. Menüden istediğin işlemi seç:

```
[1] Telemetri & Gizlilik
[2] Bloatware Kaldır
[3] Performans Optimizasyonu

```

---

### ⚠️ Uyarı

* Bazı işlemler **geri alınamaz**

## 🇬🇧 English

### 📌 About Project

**Win11 Debloat Tool** is a CLI-based utility for Windows 11 that:

* Removes unnecessary apps
* Disables telemetry and tracking
* Optimizes system performance
* Controls Windows Update behavior

Requires **Administrator privileges**.

---

### ⚙️ Features

#### 🔒 Telemetry & Privacy

* Disables telemetry services
* Disables Cortana
* Turns off Advertising ID
* Disables Activity History
* Disables Location tracking

#### 🧹 Bloatware Removal

* Removes Xbox apps
* Removes Teams, Skype
* Removes Candy Crush, TikTok, etc.
* Cleans preinstalled Microsoft apps

#### ⚡ Performance Optimization

* Enables High Performance power plan
* Disables unnecessary services
* Turns off animations
* Improves system responsiveness

#### 🔄 Windows Update Control

* Disables auto restart
* Disables driver updates
* Disables Store auto updates

---

### ▶️ Usage

1. Run as **Administrator**
2. Select from menu:

```
[1] Telemetry & Privacy
[2] Remove Bloatware
[3] Performance Optimization
[4] Windows Update Settings
[5] Apply All
```

---

### ⚠️ Disclaimer

* Some actions are **irreversible**
* Modifies system settings
* **Restart recommended** after usage
